Click to Add Gravitational Objects










Information: This is a simulation of how gravitational objects (planets, stars) would interact using Newton's law of universal gravitation. This is a famous mathematical problem because there exists no exact solution (full calculation of all N positions as a function of time). Even small changes to initial conditions can lead to very different (unpredictable) outcomes in the future. Changing the fixed radius slider and toggle (lowest slider) allows for creation of high density objects (neutron stars / black holes). Zoom with mouse scroll wheel.
